# Aurora 3D Christmas Tree

基于 Web 技术的 3D 粒子圣诞树与极光视觉交互探索项目。

## 项目简介

本项目是一个纯前端的 3D 视觉与交互实验。在传统的粒子渲染基础上，融合了 WebGL/Shader 模拟的动态极光背景，并接入了基于 WebRTC 与计算机视觉的实时手势交互控制。项目完全在浏览器客户端运行，注重视觉性能与数据隐私。

 **在线体验**: [Live Demo](https://cxzgood.github.io/aurora-christmas-tree/index.html) *(注：由于浏览器安全策略，手势识别功能需在本地 localhost 或 HTTPS 环境下授权摄像头方可使用)*

## 核心特性 (Features)

* **3D 粒子系统与 Shader 渲染**
  * 实现了动态的 3D 圣诞树粒子建模，支持平滑的“聚合 / 散开”状态转换。
  * 利用底层着色器 (Shader) 逻辑模拟出高保真的动态极光 (Aurora) 与雪花/尘埃飘落的背景效果。
* **计算机视觉手势交互**
  * 接入实时手势识别算法，将物理动作映射为 UI 指令：
    * `握拳`：粒子聚合（树形态）
    * `张开手掌`：粒子散开（星云形态）
    * `捏合`：轮播展示挂载的照片
* **本地化媒体引擎 (Zero-Server)**
  * 支持用户自定义背景音乐与照片流。
  * **隐私优先**：采用 HTML5 `IndexedDB` 将所有多媒体大文件直接持久化存储在浏览器本地沙箱中，实现零服务器数据传输，确保用户隐私绝对安全。
* **动态控制台**
  * 暴露了丰富的渲染参数（雪花数量、粒子速度、尘埃密度等），支持实时调整与重绘。
