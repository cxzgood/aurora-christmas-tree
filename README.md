# 极光圣诞树

一个带极光特效的3D圣诞树网页，支持手势控制、照片上传、音乐播放等交互功能。

## 效果预览

直接用浏览器打开 `电脑版本圣诞树.html` 即可。  
（摄像头手势需要 HTTPS 或 localhost 环境，普通打开也能用其他功能）

## 主要功能

- 3D 圣诞树粒子效果，可切换聚合/散开模式
- 动态极光背景（Shader 模拟）
- 手势识别（握拳：聚合模式，张开手掌：散开模式，捏合：切换展示照片）
- 上传自己的照片，照片会挂在树上或散开成星云
- 上传背景音乐并播放
- 调整雪花数量、大小、速度，调整尘埃粒子密度
- 自定义祝福文字（字体、大小、颜色）
- 全屏、隐藏界面、拖拽标题

## 文件说明

- `电脑版本圣诞树.html` – 完整版，包含所有功能
- `手机版本.html` – 简化版，适合手机触摸操作
- `template.c` – 无关的备份文件，可忽略

## 隐私说明

所有上传的照片和音乐都存储在浏览器本地（IndexedDB），不会发送到任何服务器。换设备或清缓存会丢失数据。

## 运行方式

双击 `电脑版本圣诞树.html` 即可在浏览器中运行。  
若需要使用摄像头手势识别，请通过本地服务器（如 `npx http-server`）或部署到 HTTPS 环境打开。

## 已知问题

- 极光效果在部分老旧显卡上可能性能下降
- 手势识别在强背光环境下可能不准
